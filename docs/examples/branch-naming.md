# Example: Branch Naming

```text
feat/search-filters
fix/session-expiry
docs/update-release-notes
chore/dependabot-config
ci/add-node-tests
security/rotate-deploy-token
```

Required:

- Use lowercase.
- Use ASCII hyphens.
- Use one purpose per branch.
- Avoid names such as `misc`, `stuff`, `final`, or `new-changes`.

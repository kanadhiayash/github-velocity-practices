# Example: Pull Request Summary

```text
Summary:
Add dependency update policy and Dependabot checklist.

Facts:
- Repository currently has no dependency review checklist.
- Public repositories can use Dependabot alerts.

Assumptions:
- Maintainers want low-noise dependency updates.

Unknowns:
- Exact package ecosystems used by downstream repositories.

Risks:
- Too many automated updates may create review fatigue.

Validation:
- Markdown links checked.
- Public copy scan passed.
```

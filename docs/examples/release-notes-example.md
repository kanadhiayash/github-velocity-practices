# Example: Release Notes

```text
Version: v0.1.0
Date: YYYY-MM-DD

Summary:
Initial public manual scaffold.

Changes:
- Added branch strategy.
- Added pull request review protocol.
- Added CI/CD gate guidance.

Security notes:
- No live credentials included.
- Workflow permissions set to read-only by default.

Migration notes:
- None.

Known issues:
- Stack overlays may need project-specific expansion.

Rollback plan:
- Revert the release tag or publish a patch release.

Verification:
- Markdown checks passed.
```

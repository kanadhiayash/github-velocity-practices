# Release Readiness Checklist

## Required

- [ ] Release version selected.
- [ ] Release branch or tag source verified.
- [ ] Required checks passed.
- [ ] Release notes completed.
- [ ] Security notes completed.
- [ ] Migration notes completed.
- [ ] Known issues listed.
- [ ] Rollback plan completed.
- [ ] Post-release verification defined.

## Recommended

- [ ] Release candidate tested.
- [ ] Artifact integrity checked.
- [ ] Deployment environment protected.
- [ ] Monitoring or smoke test plan ready.

# New Repository Velocity Checklist

## Required

- [ ] `main` protected or covered by a ruleset.
- [ ] Pull requests required before merge.
- [ ] Required status checks configured.
- [ ] Force pushes blocked on protected branches.
- [ ] Pull request template added.
- [ ] `SECURITY.md` added.
- [ ] CI workflow added.
- [ ] Workflow permissions set explicitly.
- [ ] Dependabot alerts enabled where available.
- [ ] Secret policy documented.

## Recommended

- [ ] Dependabot version updates configured.
- [ ] Code scanning configured where appropriate.
- [ ] Release notes template added.
- [ ] Rollback template added.
- [ ] Branch naming policy documented.

## Optional

- [ ] Code owners added.
- [ ] Signed commits required.
- [ ] Tag protection configured.
- [ ] Deployment environments configured.

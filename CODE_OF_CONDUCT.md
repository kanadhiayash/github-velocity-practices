# Code of Conduct

## Standard

This repository expects respectful, specific, and constructive participation.

## Required behavior

- Discuss ideas, decisions, and tradeoffs, not personal identity.
- Explain disagreement with evidence or reasoning.
- Keep feedback actionable.
- Avoid harassment, insults, threats, or discriminatory language.
- Respect security reporting boundaries.

## Unacceptable behavior

- Publishing private information.
- Posting secrets, credentials, or exploit instructions outside the security process.
- Using issues or pull requests for personal attacks.
- Repeatedly derailing technical discussion.

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or block contributors who violate these standards.
